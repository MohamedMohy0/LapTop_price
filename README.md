#  Laptop Price Prediction

A Machine Learning project that predicts laptop prices based on different hardware and software specifications using regression models and data preprocessing techniques.

Repository: :contentReference[oaicite:0]{index=0}

---

#  Overview

This project aims to build an intelligent laptop price prediction system using Machine Learning.

The workflow includes:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation
- Streamlit Deployment

The application allows users to enter laptop specifications and get an estimated price prediction instantly.

#  Installation

## 1️ Clone Repository

```bash
git clone https://github.com/AbdoAtef2000/LapTop_price.git
cd LapTop_price
```

## 2️ Install Requirements

```bash
pip install -r requirements.txt
```

## 3️ Run Streamlit App

```bash
streamlit run app.py
```
